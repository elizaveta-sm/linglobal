## Installation

The project requires node js and git.

### Clone the website using
```npm
git clone https://github.com/elizaveta-sm/linglobal
```

### Install the frontend using

```bash
  cd client
  npm i
```

### Run using

```bash
  npm run dev
```

### In the browser open

```bash
  http://localhost:5173/
```
